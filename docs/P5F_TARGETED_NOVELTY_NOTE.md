# P5F Targeted Novelty Note

Search updated 2026-07-21. Privacy-utility analysis itself is not novel.
Recent work already studies dimension and neighborhood size in locally private
GNNs, instance-specific utility for private network principal components, and
privacy/utility improvements through edge refinement. The admissible claim is
therefore narrower: an audited frontier for ordinary-graph federated link
prediction under add/remove edge-DP, contrasting individually visible client
messages with ideal secure aggregation and requiring inference-closed scores.

Primary neighboring sources inspected:

- Going Deeper into Locally Differentially Private Graph Neural Networks:
  https://openreview.net/forum?id=2aKHuXdr7Q
- Differentially Private and Scalable Estimation of the Network Principal
  Component: https://openreview.net/forum?id=V0BjWbrAYC
- Differentially Private Decoupled Graph Convolutions:
  https://openreview.net/forum?id=dd3KNayGFz
- EdgeRefine (2026 preprint): https://arxiv.org/abs/2607.08659
- Differentially Private Link Prediction With Protected Connections:
  https://arxiv.org/abs/1908.04849

No "first privacy-utility frontier" claim is admissible.
