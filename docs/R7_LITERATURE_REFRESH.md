# R7 Literature Refresh

Search date: 2026-07-24 (Asia/Shanghai).

## Scope

The refresh repeated exact-task and near-task searches for federated graph
link prediction, edge differential privacy, decentralized graph learning, and
private recommendation. Publisher pages and primary papers were used for
technical classification.

## New high-risk neighbor

Li, Wang, and Luo, [A Two-Phase Semantic Fusion Framework for
Privacy-Preserving Federated Heterogeneous Graph
Learning](https://doi.org/10.1007/s44443-026-00704-8), J. King Saud Univ.
Comput. Inf. Sci., 2026, is now included as FedHGPP.

The publisher article states:

- federated heterogeneous graph learning across institutional silos;
- two-stage structural perturbation with claimed pure edge-level DP;
- encrypted cross-graph embedding aggregation; and
- recommendation evaluation using HR@10 and NDCG@10 on three datasets.

This work invalidates any broad novelty claim based only on combining
federation, edge DP, and link-oriented recommendation. It does not disclose a
private target-domain utility certificate, no-harm fallback policy,
finite-holdout certification theorem, or the same inference-closed branch
contract. The R7 manuscript now cites it in the landscape table,
decentralized/federated discussion, private LP discussion, and introduction.

## Other refreshed neighbors

- [Efficient and Privacy-Preserved Link Prediction via Condensed
  Graphs](https://doi.org/10.1016/j.eswa.2025.128937), Expert Systems with
  Applications 2026, studies graph condensation and privacy/attack tradeoffs,
  but not a federated edge-DP transcript or private deployment gate.
- [FedEGL](https://www.sciencedirect.com/science/article/pii/S1566253525011807),
  Information Fusion 2026, protects approximated node features in
  edge-assisted federated graph learning; its main task is node
  classification and its privacy object differs.
- [A Privacy-Enhancing Mechanism for Federated Graph Neural
  Networks](https://doi.org/10.3390/sym17040565), Symmetry 2025, remains the
  direct local-perturbation federated LP neighbor already cited.

## Novelty decision

No exact predecessor was found that simultaneously provides:

1. distributed ordinary interaction-edge ownership;
2. an inference-closed edge-DP structural branch and complete released
   training/certification transcript;
3. a private target-domain bounded-utility certificate;
4. a no-harm activation/fallback policy; and
5. the registered finite-population theorem and privacy-cost accounting.

Therefore R7 does not trigger the exact-predecessor stop condition. The
defensible novelty is the **private deployment certification policy**, not
federated private link prediction itself and not a new graph encoder.
